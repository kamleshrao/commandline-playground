# Terminal Tricks 

## To clear the commands from Terminal screen

#### Ubuntu 
``` 
CTRL + L 
```

#### Mac OS 
``` 
CMD + K 
```

## To create a blank file 
```
touch index.html
```

The above command will create a blank file named `index.html` in the current directory.
